## 📖 このブログについて

### コンセプト

「**Warm Minimalism**（温かいミニマリズム）」をデザイン哲学とし、pure HTML/CSSで構築された技術ブログです。

- **引き算の美学**: 必要最小限の要素で最大の表現
- **可読性**: 長文記事でも疲れない読書体験
- **温かみ**: 冷たくなりがちなミニマルデザインに人間味を

### 記事カテゴリー

- 🔥 **Burn設計** - 引き算の美学と可逆性の設計思想
- 🏗️ **アーキテクチャ** - システム設計とプラットフォーム構想
- 📔 **開発日記** - 日々の開発記録と気づき
- 🔧 **トラブルシューティング** - 遭遇した問題と解決方法

### 技術スタック

```
フロントエンド: Pure HTML5 + CSS3 + Vanilla JavaScript
デザイン: Warm Minimalism
フォント: Noto Sans JP, Fira Code
ホスティング: GitHub Pages
```

**フレームワークなし、ビルドツールなし。**  
シンプルさを追求した結果、最も基本的な技術に回帰しました。

## 🎨 特徴

### 1. カテゴリーフィルター

記事を4つのカテゴリーで絞り込み可能。JavaScriptによる動的フィルタリングで、ユーザーが興味のあるコンテンツを素早く見つけられます。

### 2. ダークモード対応

目に優しいダークモードを実装。LocalStorageで設定を記憶します。

### 3. レスポンシブデザイン

デスクトップからモバイルまで、あらゆる画面サイズで快適な読書体験を提供。

### 4. SEO最適化

適切なメタタグ、セマンティックHTML、構造化データで検索エンジン最適化。

## 🚀 使い方

### ローカルで閲覧

```bash
# リポジトリをクローン
git clone https://github.com/mochipj2025/mochisura_blog.git

# ディレクトリに移動
cd mochisura_blog

# index.htmlをブラウザで開く
# または、シンプルなHTTPサーバーを起動
python -m http.server 8000
# → http://localhost:8000 にアクセス
```

### 新しい記事の追加

1. `articles/` ディレクトリに新しいHTMLファイルを作成
2. `index.html` に記事カードを追加
3. 適切な `data-category` 属性を設定

## 🔥 Burn設計について

このブログで語られる「Burn」とは、単なる技術手法ではなく、設計哲学です。

### 核心原則

1. **引き算の美学** - 本質を見極め、余分を削ぎ落とす
2. **可逆性** - いつでも元に戻せる設計
3. **美しく焼き上げる** - シンプルさの中に美を見出す

詳しくは[Burn設計哲学](articles/burn-philosophy.html)の記事をご覧ください。
